# usina-play


### 🛠 Tecnologias
- [Node.js] versão 20.12.2,
-[Angular.js] versão 17,
-[ionic] versão 8

### 🎲 Rodando o ionic 
# Clone este repositório
$ git clone <https://github.com/tgmarinho/nlw1>

# Instale as dependências
$ npm install

# Execute a aplicação em modo de desenvolvimento
$ ionic serve
